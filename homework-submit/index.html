Submitting homework

November 05, 2020
Course Logistics

Homework submissions will be accomplished through Canvas and GitHub. In Canvas, I would like the final code, instructions to run the code and a final report. The final report should be typed neatly, in proper English, in Rmarkdown. The code should be properly commented and adhere to both Reproducible Research and Good Programming Practices. Included with the instructions to run the code should be a link to a DockerHub image that can be pulled via Singularity and run on TinkerCliffs.

For discussions of Reproducible Research and Good Programming Practices, see:

http://adv-r.had.co.nz/Style.html or https://privefl.github.io/advr38book/good-practices.html or from Google: https://google.github.io/styleguide/Rguide.html

For Reproducible Research: http://www.biostat.jhsph.edu/~rpeng/research.html https://www.pnas.org/content/112/6/1645
